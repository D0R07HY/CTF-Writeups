# Ancient Artifact (Reverse, 400pts)

**Flag:** `DELTA{wh4t_4_gr33k_3nc0d3r_t3ch_hUh_01d_t3ch_1snt_1t?}`

## โจทย์

ข้อมูลถูกเข้ารหัสหลายชั้น โดยชั้นแรกใช้สัญลักษณ์กรีกแทนตัวเลข ก่อนจะโดนแปลงต่ออีกสองชั้น

## ขั้นตอน

1. ทำ mapping สัญลักษณ์กรีก -> ตัวเลข `0-9`
2. อ่านข้อมูลทีละ 3 ตัวอักษรเพื่อแปลงกลับเป็น byte
3. นำผลลัพธ์ไปย้อน ROT13
4. Base64-decode อีกชั้น
5. ได้ plaintext ที่มี flag

แนวทาง:

```python
nums = ["".join(digits[ch] for ch in chunk) for chunk in triples]
stage1 = bytes(int(n) for n in nums)
```

## หลักการ

โจทย์ที่มีหลายชั้นควรแกะเป็น pipeline ให้ชัดว่าแต่ละชั้นรับอะไรและคืนอะไร ไม่อย่างนั้นจะสับสนว่าข้อมูลตอนนี้อยู่ในรูปแบบไหน

## สรุป

- แกะทีละชั้น อย่าพยายามข้าม
- เมื่อชั้นนอกสุดเป็นแค่ symbol encoding งานที่เหลือจะเริ่มชัดขึ้นเร็ว
