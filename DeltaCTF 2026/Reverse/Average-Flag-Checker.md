# average flag checker (Reverse, 400pts)

**Flag:** `DELTA{r3v_1s_d3f1n1t3ly_fun_4s_h3ll_0xd34db33f}`

## โจทย์

ไบนารีตรวจ flag โดยใช้การแปลง byte หลายชั้นและสลับตำแหน่งข้อมูล

## ขั้นตอน

1. เปิดไบนารีใน disassembler
2. หา logic ที่เปรียบเทียบ input กับค่าคงที่
3. ดึงตารางแปลงค่าและ permutation ออกจาก `.rodata`
4. เขียนสคริปต์ย้อน transform ทีละขั้น
5. กู้สตริง flag เดิมกลับมา

## หลักการ

flag checker จำนวนมากไม่ได้ซับซ้อนทางคณิตศาสตร์ แต่ซับซ้อนเพราะพยายามทำให้ลำดับการแปลงอ่านยาก

## สรุป

- หา "ค่าคงที่ปลายทาง" และ "วิธีแปลง" ให้ครบ
- จากนั้นเขียนตัว reverse กลับแทนการเดา input ตรง ๆ
