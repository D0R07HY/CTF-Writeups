# Checking Flags (Reverse, 400pts)

**Flag:** `DELTA{x73a_c1ph3r_1s_v3ry_c00l!}`

## โจทย์

ไบนารีฝังบล็อกข้อมูลเข้ารหัสและคีย์เอาไว้ จากนั้นถอดรหัสเพื่อตรวจว่า input ถูกหรือไม่

## ขั้นตอน

1. หา function ตรวจ flag ในไบนารี
2. สังเกตรูปแบบรอบการคำนวณและค่าคงที่
3. ระบุว่าเป็นการเข้ารหัสสไตล์ TEA
4. ดึง key และ ciphertext ที่ hardcode ไว้ออกมา
5. เขียนตัวถอดรหัสแล้วกู้ plaintext เป็น flag

## หลักการ

เมื่อโปรแกรมตรวจ flag ด้วยการถอดรหัสข้อมูลที่ฝังอยู่ เราสามารถทำแบบเดียวกันนอกโปรแกรมได้ และมักจะง่ายกว่าการพยายาม brute-force input

## สรุป

- มองหาค่าคงที่ของ TEA และรอบการบวก/shift/xor
- ดึง key ออกมาแล้วถอด offline จะเร็วที่สุด
