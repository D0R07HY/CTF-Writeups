# Find The Flag (Misc)

**Flag:** `DELTA{th3_r34l_fl4g_1s_h3r3_c0ngr4ts_0n_f1nd1ng_1t}`

## โจทย์

ไฟล์หรือข้อความที่โจทย์ให้มามีข้อมูลรบกวนจำนวนมาก โดยซ่อน flag เอาไว้ในเนื้อหาด้านใน

## ขั้นตอน

1. เปิดไฟล์ที่แนบมากับโจทย์
2. ค้นหารูปแบบของ flag เช่น `DELTA{`
3. ดึงข้อความที่อยู่ในรูปแบบ flag ออกมา

ตัวอย่างคำสั่งที่ใช้ได้:

```bash
grep -o "DELTA{[^}]*}" challenge.txt
```

## หลักการ

โจทย์แนวนี้ไม่จำเป็นต้อง reverse หรือ exploit อะไร แค่กรองข้อมูลให้เร็วและมองหารูปแบบที่ชัดเจน

## สรุป

- มองหารูปแบบของ flag ก่อน
- ใช้เครื่องมือ search เช่น `grep`, `rg`, หรือ editor search ช่วยได้มาก
