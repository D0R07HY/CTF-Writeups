# Robots (Misc)

**Flag:** `DELTA{r0b0ts_4r3_r34l_frfr}`

## โจทย์

โจทย์เว็บง่าย ๆ ที่ชี้ให้ดูจุดซ่อนข้อมูลในเส้นทางของเว็บ

## ขั้นตอน

1. เปิดไฟล์ `robots.txt` ของเว็บไซต์
2. อ่าน path ที่ถูกซ่อนไว้จาก crawler
3. เข้า path นั้นต่อเพื่อดูข้อมูลที่ไม่ถูกลิงก์จากหน้าเว็บหลัก
4. พบ flag

ตัวอย่าง:

```text
/robots.txt
```

## หลักการ

`robots.txt` เป็นไฟล์ที่ใช้บอก crawler ว่า path ไหนไม่ควรเก็บดัชนี แต่ไม่ได้เป็นการป้องกันการเข้าถึงจริง จึงมักถูกใช้เป็นที่ซ่อน hint หรือไฟล์สำคัญใน CTF

## สรุป

- เว็บ CTF ควรเช็ก `robots.txt` เสมอ
- คำว่า "ห้ามเข้า" สำหรับ crawler ไม่ได้แปลว่า "คนเข้าไม่ได้"
