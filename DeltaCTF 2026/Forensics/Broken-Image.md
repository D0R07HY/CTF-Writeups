# Broken Image (Forensics, 436pts)

**Flag:** `DELTA{4v3r4g3_st3g4n0gr4phy_ctf_3z}`

## โจทย์

ไฟล์ภาพ JPEG เปิดไม่ได้ เพราะ header ด้านหน้าถูกทำให้เสีย

## ขั้นตอน

1. เปิดไฟล์ด้วย hex viewer
2. หา marker ที่บ่งบอกว่าโครงสร้าง JPEG จริงยังอยู่ เช่น `JFIF`
3. เติม SOI และ APP0 header ที่ถูกต้องกลับเข้าไป
4. เขียนไฟล์ใหม่แล้วเปิดภาพ
5. อ่าน flag จากภาพที่ซ่อมแล้ว

แนวคิดของการซ่อม:

```python
repaired = b"\xff\xd8\xff\xe0" + data[jfif_at - 2:]
```

## หลักการ

หลายครั้งไฟล์ที่ดูเหมือนเสียจริง ๆ แค่ header พัง แต่ payload ด้านในยังสมบูรณ์อยู่

## สรุป

- ถ้าภาพเปิดไม่ได้ ให้ดู magic bytes ก่อน
- ซ่อม header สำเร็จแล้วมักจะเห็นคำตอบทันที
